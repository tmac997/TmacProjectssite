Projects

  • Nmap and Wireshark Network Scanning Project

    This project involved scanning a corporate network using Nmap to identify open ports, services, and vulnerabilities, followed by analyzing traffic using Wireshark. The results provided valuable insights into network topology and security risks. During the scan, multiple machines running on different operating systems, including Windows and Linux, were identified. The analysis discovered open ports, such as port 445, which were linked to vulnerabilities like CVE-2022-0918 and CVE-2002-0283. In addition, vulnerabilities in SSH services, including CVE-2024-38510, were analyzed, highlighting potential privilege escalation risks. The network's star topology indicated centralized administration, streamlining network management.
